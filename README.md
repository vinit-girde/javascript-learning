# Learning JavaScript

This repository, [https://github.com/vinit-girde/learning-javascript](https://github.com/vinit-girde/javascript-learning), documents my journey of learning and practicing JavaScript. It's a collection of code examples, exercises, and projects that I've created to solidify my understanding of the language.

## About This Repository

This repository serves as a personal learning space for JavaScript. You'll find:

- **Code Examples:** Snippets illustrating various JavaScript concepts, from basic syntax to advanced features.
- **Exercises:** Solutions to coding challenges and exercises from online resources or books.
- **Small Projects:** Practical applications of JavaScript, demonstrating how to build interactive web elements or simple applications.
- **Documentation:** Notes and explanations of key concepts and techniques.
- **Exploration:** experiments with new Javascript features.

## What You'll Find Inside

- **Folders organized by topic:** You'll likely find folders for topics like:
  - `basics/`: Covering fundamental syntax, variables, data types, operators.
  - `Iterations/`: Exploring iterators in JS.
  - `Events/`: Working with events listeners and exploring Browser API's functions like setTimeout and interval and Event Propagation .
  - `Control Flows/`: Control flows statements and truthy values.
  - `dom/`: Interacting with the Document Object Model (DOM).
  - `Advance/`: Concept like call,apply,bind,closure,currying,getter,setters,pototype and promises.
- **`README.md` files (within folders):** More detailed explanations and instructions for specific topics.

## Technologies Used

- **JavaScript (ES6+):** The core programming language.
- **Node.js:** For running JavaScript code outside the browser (if applicable).
- **HTML/CSS:** For basic web pages to demonstrate DOM manipulation.
- **Browser Developer Tools:** For debugging and inspecting JavaScript code.

## How to Run the Code

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/vinit-girde/learning-javascript.git](https://www.google.com/search?q=https://github.com/vinit-girde/learning-javascript.git)
    cd learning-javascript
    ```

2.  **For Node.js examples:**

    - Navigate to the directory containing the `.js` file.
    - Run the file using Node.js: `node filename.js`

3.  **For browser-based examples:**

    - Open the corresponding HTML file in your web browser.
    - Open the browser's developer console to see the output or interact with the code.
